Binary Search Tree

A binary search tree (BST) arranges its elements based on a specific order: the value of each left node must be smaller than its parent node, and the value of each right node must be greater than its parent node.

Path Finding Algorithms

Pathfinding algorithms aim to solve shortest and most efficient path problems in graph theory, finding the optimal path between a starting point and an ending point based on predefined criteria.

Sorting Algorithms

Sorting algorithms are used for arranging elements in a data structure in a specific order, such as ascending or descending, making data organization and searching more efficient.
